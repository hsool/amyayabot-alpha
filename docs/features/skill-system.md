# 스킬과 음성 명령 완벽 가이드

에이미와 상호작용하는 여러 가지 게임과 기능들을 "스킬"이라고 합니다. 이 문서는 각 스킬이 어떻게 동작하고, 어떻게 사용하는지 설명합니다.

## 스킬 시스템 개요

### 스킬을 쓰는 방법

스킬은 **2가지 방법**으로 시작할 수 있습니다:

#### 1. 음성 명령 (가장 추천)

호명 후 자연스럽게 말하기:

```
당신: "에이미야, 투표 시작할래?"
에이미: "좋아! 항목을 말씀해주세요"
```

#### 2. 채팅 명령 (직접 입력)

채팅창에 특정 명령어 입력:

```
!투표 1        (투표 스킬 - 1번 항목에 투표)
!참여          (추첨 스킬 - 추첨 참여)
```

### 음성 명령의 동작 원리

```
1. 호명: "에이미야!"
   ↓
2. 대화 모드 진입
   ↓
3. 당신의 발화에서 스킬 키워드 감지
   ↓
4. 해당 스킬 자동 선택
   ↓
5. AI가 필요한 정보 자동으로 묻기
   ↓
6. 당신의 답변 기반으로 스킬 실행
```

## 스킬 종류와 사용법

### 1. 투표 (Vote)

**목적**: 시청자들과 주제로 투표하기

**키워드 인식**: "투표", "투도", "투보"

#### 사용 방법

**음성 명령**:
```
당신: "에이미야, 투표를 한 번 해봐!"
에이미: "좋아! 투표 항목이 뭔데?"
당신: "치킨, 피자, 햄버거"
에이미: "오케이! 투표를 시작합니다!"

시청자들이 채팅에서:
!투표1    (치킨에 투표)
!투표2    (피자에 투표)
!투표3    (햄버거에 투표)
또는
!투표 치킨 (치킨에 투표)
```

**채팅 명령**:
```
!투표 1        투표 옵션 1 선택
!투표 2        투표 옵션 2 선택
!투표결과      투표 즉시 종료 및 결과 발표 (스트리머만)
```

#### 투표의 종류

**선택형 투표** (항목을 정해서 시작)
- 항목: 치킨 vs 피자 vs 햄버거 (정해진 보기)
- 시청자는 숫자나 항목명으로 투표

**자유 키워드 투표** (항목 없이 시작)
- 항목을 정하지 않고 시작
- 시청자가 자유롭게 입력 가능
- 예: "!투표 피자" → "피자"에 한 표

#### 투표 설정

당신이 말할 수 있는 것들:

```
"투표를 3분으로 설정해줄래?"
→ 투표 시간 제한 설정

"투표 결과를 숨겨줄래?"
→ 진행 중 결과를 비공개 처리 (투표 종료 시만 공개)

"투표 진행 현황을 알려줄래?"
→ 현재 투표 현황 확인
```

#### 투표 화면

오버레이에 표시되는 내용:
- 각 항목별 표 수 (또는 총 투표 수만 표시)
- 진행 중 실시간 업데이트
- 투표 종료 시 우승자 표시

### 2. 추첨 (Raffle)

**목적**: 시청자 중 랜덤하게 당첨자 선출

**키워드 인식**: "추첨", "뽑기", "시청자참여"

#### 사용 방법

**음성 명령**:
```
당신: "에이미야, 추첨을 한 번 해볼까?"
에이미: "좋아! 어떻게 할까요?"
당신: "최근 채팅 사람들 중에서 뽑아"
에이미: "오케이! 지금 추첨합니다!"
```

**채팅 명령**:
```
!참여    시청자가 추첨에 참여 (참여 모드가 열렸을 때만)
```

#### 추첨의 2가지 방식

**방식 1: 즉시 추첨 (Instant Draw)**
- 최근 채팅에 참여한 사람들 중에서 즉시 뽑기
- 가장 간단하고 빠름
- 예: "방금 채팅한 사람 중에서 뽑아줄래?"

**방식 2: 참여 모드 (Participation Mode)**
- 시청자가 "!참여" 명령으로 등록
- 등록된 사람들 중에서 뽑기
- 더 공정하고 의도적
- 예: "참여를 원하시는 분들은 !참여 를 입력해주세요"

#### 당첨자 처리

```
추첨 단계:
1. 슬롯 애니메이션 (당첨자 결정, 화면에 이름들 회전)
2. 당첨자 발표
3. 당첨자가 채팅에 응답 대기 (타이머)
4. 응답 완료 → 축하 메시지
```

당신이 당첨자 관리를 위해 말할 수 있는 것:

```
"당첨자가 응답할 시간을 1분으로 줄래"
→ 당첨자 응답 대기 타이머 설정

"당첨자 대기를 취소해줄래"
→ 현재 당첨자 대기 상태 해제
```

#### 추첨 화면

오버레이에 표시:
- 당첨자 결정 중 (슬롯 애니메이션)
- 당첨자 이름 표시
- 당첨자 응답 대기 중일 때 카운트다운

### 3. 룰렛 (Roulette)

**목적**: 항목 중 하나를 랜덤하게 선택

**키워드 인식**: "룰렛", "룰래", "룰렡"

#### 사용 방법

**음성 명령**:
```
당신: "에이미야, 룰렛을 돌려볼까?"
에이미: "좋아! 룰렛에 들어갈 항목을 말씀해주세요"
당신: "치킨, 피자, 햄버거, 라면"
에이미: "좋아, 지금 돌릴게!"

(휠이 회전하면서 하나를 선택함)
```

#### 룰렛의 특징

**가중치 (Weight)**: 항목마다 당첨 확률을 다르게 설정 가능

```
당신: "피자는 3배 확률로 해줄래?"
→ 일반 항목보다 피자가 3배 자주 나옴
```

**항목 추가/제거**: 룰렛 진행 중에도 변경 가능

```
"룰렛에 국수를 추가해줄래"
"피자를 빼줄래"
```

#### 룰렛 화면

오버레이에 표시:
- 회전하는 원형 휠
- 각 항목 표시
- 3~4초 회전 후 당첨 항목 표시

### 4. 도네이션 투표 (Donation Vote)

**목적**: 후원 메시지를 통한 투표 (금액 기반)

**키워드 인식**: "도네이션투표", "후원투표", "도네투표"

#### 사용 방법

**음성 명령**:
```
당신: "에이미야, 도네이션 투표를 시작할래?"
에이미: "좋아! 투표 항목을 2개 이상 말씀해주세요"
당신: "오늘 먹을 음식을 정하자. 치킨, 피자, 햄버거"
에이미: "좋아! 도네이션 투표를 시작합니다!"

시청자들이 후원 메시지로:
후원: 5,000원 - "투표2" (피자에 투표)
후원: 10,000원 - "투표1" (치킨에 투표)
```

#### 도네이션 투표의 특징

**최소 후원 금액**: 투표에 참여하기 위한 최소 금액 (기본: 1,000원)

```
당신: "최소 5,000원 이상만 투표로 인정해줄래?"
→ 5,000원 미만 후원은 투표로 처리되지 않음
```

**복수 투표**: 금액에 따라 여러 표를 얻을 수 있음

```
최소 금액: 1,000원
시청자 후원: 3,000원 (3표 계산)

복수투표 OFF: 3,000원 = 1표 (금액 상관없이 1표)
복수투표 ON:  3,000원 = 3표 (금액 / 최소금액)
```

**익명 도네이션**: 익명 후원자도 투표 가능 여부

```
당신: "익명 후원자도 투표 가능하게 해줄래?"
→ 익명 후원 메시지도 투표로 인정
```

#### 도네이션 투표 화면

오버레이에 표시:
- 각 항목별 투표 수 (또는 가중치)
- 후원 금액 기반 실시간 업데이트
- 투표 종료 시 우승자 발표

### 5. 채팅 요약 (Chat Summary)

**목적**: 최근 채팅을 AI가 요약해서 읽어주기

**키워드 인식**: "채팅요약"

#### 사용 방법

**음성 명령**:
```
당신: "에이미야, 지금까지 채팅을 요약해줄래?"
에이미: (최근 채팅을 읽고 요약한 후 TTS로 전달)

"지난 10분간 시청자분들이 치킨 vs 피자 투표에 참여했고,
대부분 치킨을 선호했습니다. 또한 라면 추천글도 몇 개 있었습니다."
```

#### 요약 방식

**즉시 요약**: 지금 바로 최근 채팅 요약

```
"지금 채팅 내용을 정리해줄래?"
```

**주기적 요약**: 일정한 시간 간격으로 자동 요약

```
"5분마다 채팅을 요약해줄래?"
→ 5분 주기로 자동 요약 시작

"요약을 멈춰줄래?"
→ 주기적 요약 종료
```

**요약 말풍선**: 요약 내용을 말풍선으로도 표시할지 여부

```
"요약을 말풍선으로도 보여줄래?"
→ TTS 음성 + 말풍선 표시
(기본: TTS 음성만, 말풍선 없음)
```

#### 채팅 요약 화면

오버레이에 표시:
- 음성으로만 전달됨 (TTS)
- 설정에 따라 말풍선도 표시 가능

### 6. 방송 제어 (Broadcast Control)

**목적**: 방송 제목, 카테고리, 태그를 AI가 자동으로 변경

**키워드 인식**: "방송제목", "방송카테", "방송태그", "방제"

#### 사용 방법

**음성 명령**:
```
당신: "에이미야, 방송 제목을 '오늘은 게임데이'로 바꿔줄래?"
에이미: "좋아! 제목을 변경했어"
→ Chzzk 방송 제목이 자동으로 변경됨

당신: "방송 카테고리를 '게임'으로 설정해줄래?"
에이미: "오케이! 카테고리를 '게임'으로 변경했습니다"
```

#### 변경할 수 있는 항목

| 항목 | 예시 | 설명 |
|------|------|------|
| 제목 | "재미있는 게임 라이브" | Chzzk 방송 제목 변경 |
| 카테고리 | "게임", "하이라이트", "토크" | 방송 카테고리 변경 |
| 태그 | ["게임", "초보자", "친화적"] | 라이브 태그 추가/변경 |

#### 방송 정보 조회

```
당신: "지금 방송 정보가 뭔데?"
에이미: "제목: 오늘은 게임데이, 카테고리: 게임, 태그: [게임, 초보자]"
```

### 7. 설정 변경 (Settings Control)

**목적**: 봇의 동작 방식을 음성으로 조정

**키워드 인식**: "설정변경", "세팅변경", "네설정", "봇설정", "관련설정"

#### 사용 방법

**음성 명령**:
```
당신: "에이미야, 반응 간격을 2초로 해줄래?"
에이미: "좋아! 반응 간격을 2초로 변경했어"
→ 봇이 채팅에 덜 자주 반응하기 시작함

당신: "대화 모드 최대 턴을 15로 늘려줄래?"
에이미: "오케이! 대화 턴을 15로 설정했습니다"
```

#### 변경 가능한 설정들

| 설정 | 범위 | 설명 |
|------|------|------|
| 반응 간격 | 1~120초 | AI가 채팅에 반응하기까지의 최소 시간 |
| 말풍선 표시 시간 | 1~60초 | 말풍선이 화면에 표시되는 시간 |
| 하트비트 (자동 발화) | ON/OFF | 침묵할 때 자동으로 말할지 여부 |
| 하트비트 주기 | 10~300초 | 자동 발화 반복 간격 |
| TTS 음성 | ON/OFF | 음성 출력 켜기/끄기 |
| 말풍선 표시 | ON/OFF | 말풍선 표시 켜기/끄기 |
| 대화 모드 최대 턴 | 1~50 | 대화가 몇 턴까지 지속될지 |
| 방해금지모드 | ON/OFF | 침묵 시간에도 반응하지 않음 |

## 채팅 명령 전체 목록

### 공통 명령

| 명령 | 사용자 | 설명 |
|------|--------|------|
| `!투표 <옵션>` | 누구나 | 진행 중인 투표에 참여 |
| `!투표결과` | 스트리머만 | 투표 즉시 종료 및 결과 발표 |
| `!참여` | 누구나 | 진행 중인 추첨에 참여 |

### 명령어 예시

```
투표 참여:
!투표 1         (1번 항목)
!투표 2         (2번 항목)
!투표 치킨      (항목명으로 직접)

투표 결과 발표:
!투표결과       (즉시 투표 종료 및 결과 발표)

추첨 참여:
!참여           (추첨 참여 모드가 열렸을 때 입력)
```

## 스킬이 작동하지 않을 때

| 증상 | 원인 | 해결책 |
|------|------|--------|
| 음성 명령으로 스킬이 시작 안 됨 | 호명 감지 실패 또는 키워드 미인식 | 호명 감지 가이드 참조 |
| 채팅 명령 (!투표 등)이 작동 안 함 | 채팅 명령 비활성화됨 | 설정에서 `chat_commands.enabled: true` 확인 |
| 투표/추첨 실행 중 다른 스킬 못 시작 | 동시 실행 불가 | 진행 중인 스킬 종료 후 새 스킬 시작 |
| Chzzk 연결 끊어짐 | 방송 제어 스킬 실패 | Chzzk 연결 상태 확인 |

## 스킬 우선순위와 동시성

### 동시에 실행할 수 없는 것들

투표, 추첨, 룰렛, 도네이션 투표는 **한 번에 하나씩만** 실행됩니다.

```
진행 중: 투표
당신: "추첨도 시작해줄래?"
에이미: "지금 투표 중이에요. 먼저 투표를 끝내주세요"
```

### 채팅 요약과 다른 기능의 조합

채팅 요약은 다른 스킬과 함께 실행할 수 있습니다:

```
투표 진행 중 → 동시에 주기적 채팅 요약도 가능
```

## 스킬 사용 팁

1. **음성 명령은 자연스럽게**: 명령어가 아니라 대화처럼 해도 에이미가 인식합니다
2. **스킬 시작 후 단계별 응답**: 에이미의 질문에 차례대로 답변하면 됩니다
3. **채팅 명령은 정확하게**: 띄어쓰기나 숫자를 정확히 입력하세요
4. **타이머 설정**: 대부분의 스킬에서 진행 시간을 설정할 수 있습니다
