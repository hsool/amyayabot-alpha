# Settings Guide

이 섹션은 **설정 페이지를 항목별로 쉽게 설명하는 문서 묶음**이야.

## 어떻게 읽으면 좋은가?

### 빨리 확인하고 싶다면
아래 순서만 보면 돼.
1. 연결 / 자격 정보
2. 페르소나 / 대화
3. STT / 음성 제어
4. 반응 / vision
5. 출력 / TTS
6. 채팅 / 상호작용
7. 운영 / 안전

### 자세히 보고 싶다면
각 페이지는
- 무엇을 하는 설정인지
- 처음엔 어떻게 두는 게 좋은지
- 방송 중 어떤 문제가 생기면 다시 봐야 하는지
중심으로 설명해.
