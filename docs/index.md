처음 설치한 뒤 이 페이지에 왔다면,
**어디부터 읽어야 할지 헷갈리지 않게 길을 잡아주는 문서 홈**이라고 생각하면 돼.

## 이 프로젝트로 할 수 있는 것

AmyayaBot은 방송 중에:
- 스트리머 음성을 듣고 반응하고
- 시청자 채팅 흐름을 읽고 반응하고
- 후원/구독 이벤트에 반응하고
- 말풍선 / TTS / 채팅 / OBS 오버레이로 결과를 보여주고
- 필요하면 투표 / 추첨 / 룰렛 / 노래신청 같은 상호작용도 도와줘.

즉,
단순 채팅봇보다는 **방송을 같이 진행하는 AI 캐릭터 시스템**에 더 가까워.
