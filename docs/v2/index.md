# AmyayaBot v2 문서

v2 문서는 준비 중입니다.

---

## 주요 변경사항 (v1 대비)

- **ADK 기반 아키텍처** — Google Agent Development Kit으로 전환, 방송 세션 맥락 연속
- **장기기억 임베딩** — 방송 간 맥락 유지, 상호작용 패턴 학습
- **OBS WebSocket 연동** — 씬 전환 자동 감지, DND 모드 연동
- **웹 검색 도구** — Naver/Brave 검색으로 실시간 정보 제공
- **간편/상세 모드** — 비전문가 친화적 설정 UI (30개 vs 165+ 컨트롤)
- **온보딩 위자드** — 첫 실행 시 4단계 가이드
- **Settings Registry** — 음성 명령으로 대부분의 설정 조절 가능

---

자세한 설정 가이드와 사용 방법은 추후 업데이트됩니다.

v1 문서가 필요하시면 [v1 문서](../v1/)를 참고해주세요.
