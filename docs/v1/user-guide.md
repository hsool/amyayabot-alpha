## 1. 🤖 AmyayaBot이 뭔가요?

AmyayaBot은 **AI 캐릭터가 방송을 함께 해주는 프로그램**입니다.

시청자들이 채팅을 치면 AI 캐릭터가 자동으로 읽고 반응해주고, 스트리머가 이름을 부르면 대화도 할 수 있어요. 마치 함께 방송하는 AI 파트너 같은 존재입니다!

### ✨ 할 수 있는 것들

| 기능 | 설명 |
|------|------|
| 💬 채팅 반응 | 시청자 채팅을 읽고 AI가 자동으로 반응해줍니다 |
| 💝 후원 반응 | 후원이 들어오면 특별한 반응을 해줍니다 |
| 🗣️ 목소리 대화 | 스트리머가 이름을 부르면 AI가 대답합니다 |
| 🔊 TTS 음성 | AI의 반응을 음성으로 들려줍니다 |
| 💬 말풍선 | AI의 반응을 화면에 예쁘게 표시합니다 |
| 🎭 AI 캐릭터 | 장난꾸러기, 츤데레 등 개성 있는 캐릭터 설정 가능 |
| 🖼️ 아바타 | 화면에 표시되는 AI 캐릭터 이미지 |
| 🗳️ 채팅 투표 | 시청자 투표를 쉽게 진행할 수 있습니다 |
| 🎵 노래 신청 | 시청자가 채팅으로 노래를 신청할 수 있습니다 |
| 🛡️ 콘텐츠 필터 | 부적절한 주제는 AI가 반응하지 않도록 설정 |

## 2. 🛠️ 시작하기 전에 준비할 것

### 2.1 프로그램 설치

AmyayaBot을 실행하려면 두 가지 프로그램이 필요합니다.

#### 🐍 Python 설치

Python은 AmyayaBot의 핵심 엔진입니다 (백엔드 서버를 돌리는 프로그램이에요).

1. https://www.python.org/downloads/ 에 접속합니다
2. "Download Python 3.12" (또는 3.13) 버튼을 클릭합니다
3. 다운로드된 파일을 실행합니다
4. ⚠️ **중요!** 설치 화면 맨 아래에 **"Add Python to PATH"** 체크박스가 있습니다. 반드시 체크하고 설치하세요!

   > 💡 "PATH에 추가"란? 컴퓨터가 Python을 어디서든 찾을 수 있게 해주는 설정입니다. 이걸 빠뜨리면 나중에 오류가 납니다.

5. "Install Now" 클릭 → 설치 완료!

#### 🟢 Node.js 설치

> ℹ️ 빌드 버전(실행 파일)에서는 Node.js가 필요하지 않습니다. 소스 코드에서 직접 실행하는 경우에만 필요합니다.

#### ✅ 설치 확인 방법

설치가 잘 됐는지 확인해볼게요.

1. 키보드에서 **Windows 키 + R** 을 누릅니다
2. `cmd` 라고 입력하고 엔터 (명령 프롬프트가 열립니다)
3. 아래 명령어를 하나씩 입력하고 엔터를 누릅니다:

```
python --version
```
→ `Python 3.12.x` 같은 글자가 나오면 성공!

```
node --version
```
→ `v20.x.x` 같은 글자가 나오면 성공!

> ❌ "'python'은 내부 또는 외부 명령..." 같은 오류가 나온다면?
> Python 설치 시 "Add Python to PATH"를 체크하지 않은 것입니다.
> Python을 다시 설치하거나, 제어판 → 앱에서 Python을 찾아 "Modify" → "PATH 추가"를 선택하세요.

### 2.2 API 키 발급

API 키는 프로그램이 외부 서비스(치지직, Gemini AI)에 접속하기 위한 비밀번호 같은 것입니다.

#### 🎮 치지직 API 키 발급

치지직 방송 플랫폼에 연결하기 위한 키입니다.

**① 개발자 포털 가입**

1. https://developers.chzzk.naver.com/ 에 접속합니다
2. 네이버 계정으로 로그인합니다 (치지직 계정과 같은 네이버 계정을 사용하세요)

**② 애플리케이션 만들기**

1. "새 애플리케이션 만들기" (또는 "Create Application") 버튼을 클릭합니다
2. 아래와 같이 입력합니다:
   - **이름**: 아무거나 입력해도 됩니다 (예: "내 AI 봇", "방송용 봇")
   - **리디렉션 URL**: `http://localhost:8080/` (⚠️ 이 주소를 정확히 그대로 입력하세요! 끝에 `/`도 포함해서요)
   - **권한**: "채팅 읽기"와 "채팅 쓰기" 항목을 선택합니다
3. 저장/생성 버튼을 클릭합니다

**③ Client ID와 Client Secret 복사**

애플리케이션이 만들어지면 **Client ID**와 **Client Secret** 값이 보입니다.
이 두 가지를 메모장에 복사해 두세요! (나중에 설정에 입력해야 합니다)

> ⚠️ Client Secret은 한 번만 보여주는 경우가 있습니다. 꼭 복사해두세요!

**④ Channel ID 확인**

Channel ID는 내 치지직 채널의 고유 번호입니다.

1. https://chzzk.naver.com/studio 에 접속합니다
2. 왼쪽 메뉴에서 "설정" 클릭
3. URL 주소창에 보이는 숫자/문자 조합이 Channel ID입니다
   - 예: `https://chzzk.naver.com/studio/abc123def456/settings` 라면 `abc123def456` 이 Channel ID

---

#### 🤖 Gemini API 키 발급

Gemini는 Google의 AI 서비스로, AmyayaBot의 두뇌 역할을 합니다.

1. https://aistudio.google.com/apikey 에 접속합니다
2. Google 계정으로 로그인합니다 (Gmail 계정이면 됩니다)
3. "Create API Key" 버튼을 클릭합니다
4. 생성된 키를 복사해서 메모장에 저장해 두세요

> 💡 Gemini API는 무료로 사용할 수 있습니다 (일정 사용량까지는 무료입니다).

## 3. 📦 설치와 실행

### 3.1 프로그램 다운로드

**방법 1: ZIP 파일로 다운로드 (초보자 권장)**

1. AmyayaBot GitHub 페이지에 접속합니다
2. 초록색 "Code" 버튼 클릭 → "Download ZIP" 클릭
3. 다운로드된 ZIP 파일을 원하는 폴더에 압축 해제합니다
   - 예: `C:\AmyayaBot\` 또는 `바탕화면\AmyayaBot\`

**방법 2: git clone (git을 사용하는 경우)**

```
git clone https://github.com/[저장소주소]/amyayabot.git
```

---

### 3.2 실행하기

프로그램 폴더를 열면 아래와 같은 파일들이 보입니다.

#### 🪟 Windows 사용자

`start.bat` 파일을 **더블클릭**합니다.

> ❓ "Windows가 PC를 보호했습니다" 창이 뜨면?
> "추가 정보" 클릭 → "실행" 클릭하면 됩니다.

#### 🐧 Linux / 🍎 Mac 사용자

터미널을 열고 프로그램 폴더로 이동한 뒤 아래 명령어를 실행합니다:

```bash
./start.sh
```

---

### 3.3 처음 실행 시

처음 실행하면 필요한 프로그램들을 자동으로 설치합니다. **시간이 좀 걸릴 수 있어요** (보통 3~10분).
검은 창에 글자들이 쭉 올라가는 것이 정상입니다. 기다려주세요!

설치가 완료되면 아래 메시지가 나타납니다:

```
✅ Started successfully
```

이 메시지가 보이면 성공입니다! 🎉

---

### 3.4 설정 페이지 열기

프로그램이 실행된 상태에서:

1. 웹 브라우저 (크롬, 엣지 등)를 열고
2. 주소창에 `http://localhost:18300/settings` 를 입력하고 엔터를 누릅니다

   > 💡 "localhost"는 내 컴퓨터 안의 주소입니다. 인터넷이 아닌 내 컴퓨터에서 실행 중인 프로그램에 접속하는 방법이에요.

3. 화면 왼쪽에 메뉴가 보이면 성공입니다!

### 4.2 🔑 API 연결 탭

#### 이게 뭔가요?

준비해둔 API 키들을 여기에 입력합니다. 이 작업을 해야 치지직 방송과 연결되고 AI가 동작합니다.

**치지직 연결**

1. **Client ID** 입력란에 복사해둔 Client ID를 붙여넣습니다
2. **Client Secret** 입력란에 Client Secret을 붙여넣습니다
3. **Channel ID** 입력란에 Channel ID를 붙여넣습니다
4. **"연결"** 버튼을 클릭합니다
5. 브라우저에 네이버 로그인 창이 새로 열립니다 → 로그인합니다
6. 연결 상태가 **"연결됨"** 으로 바뀌면 성공!

**Gemini AI 연결**

1. **Gemini API Key** 입력란에 발급받은 키를 붙여넣습니다
2. 하단의 **"설정 저장"** 버튼을 클릭합니다

> ⚠️ 설정 저장 버튼을 꼭 누르셔야 합니다! 안 누르면 저장이 안 됩니다.

### 4.3 🐾 페르소나 탭

#### 이게 뭔가요?

AI 캐릭터의 성격과 말투를 설정하는 곳입니다. 여기를 잘 설정할수록 더 자연스럽고 재미있는 반응이 나옵니다!

**설정 방법**

1. **프리셋 선택**: 19가지 캐릭터 중 하나를 골라보세요
   - 예: 장난꾸러기, 츤데레, 다정한 도우미, 개그캐릭터 등
2. **캐릭터 이름**: AI의 이름을 설정합니다 (채팅에서 `[이름]` 형태로 표시됩니다)
3. **스트리머 이름**: 내 닉네임을 입력하면 AI가 이름으로 불러줍니다
4. **방송 분위기**: 어떤 방송인지 간략히 적어주세요 (예: "게임 방송", "잡담 방송")

> 💡 입력 내용이 많을수록 AI가 방송 상황을 더 잘 이해합니다!

**사용 예시**

> "츤데레" 프리셋에 이름을 "아먀야"로 설정하면, 시청자 채팅에 퉁명스럽지만 은근히 챙겨주는 말투로 반응합니다.

### 4.4 🖼️ 아바타 설정 탭

#### 이게 뭔가요?

방송 화면에 표시되는 AI 캐릭터 이미지를 설정합니다.

**아바타 모드 선택**

| 모드 | 특징 | 추천 대상 |
|------|------|----------|
| PNG 모드 | 감정에 따라 다른 이미지로 교체. 가장 간단! | 처음 설정하는 분 |
| SVG 모드 | 눈 깜빡임 등 세밀한 애니메이션 지원 | 좀 더 생동감을 원하는 분 |
| Live2D 모드 | 가장 자연스러운 움직임, 립싱크 지원 | 최고 품질을 원하는 분 |

**설정 방법**

1. 원하는 모드를 선택합니다
2. 이미지 파일을 업로드합니다 (PNG 모드의 경우 감정별로 여러 이미지 업로드 가능)
3. 아바타 위치와 크기는 **드래그**로 직접 조정할 수 있습니다

## 5. 📺 OBS에 연결하기

#### 이게 뭔가요?

OBS(방송 소프트웨어)에 아바타와 말풍선을 추가하는 방법입니다. 게임 화면 위에 AI 캐릭터가 떠있게 됩니다!

**연결 방법**

1. **OBS Studio**를 엽니다
2. 화면 아래 **"소스"** 영역에서 **+** 버튼을 클릭합니다
3. 목록에서 **"브라우저"** 를 선택합니다
4. **"새로 만들기"** 선택 → 이름을 입력합니다 (예: "AI 봇 오버레이")
5. 다음과 같이 입력합니다:
   - **URL**: `http://localhost:18300/overlay`
   - **너비**: `1920`
   - **높이**: `1080`
   - (방송 해상도가 다르다면 그에 맞게 조정하세요)
6. **"확인"** 클릭

아바타와 말풍선이 보이면 성공입니다! 🎉

> 💡 배경은 자동으로 투명 처리됩니다. 게임 화면 위에 올려놓으면 아바타만 보입니다.

> 💡 소스 레이어 순서: OBS에서 "AI 봇 오버레이" 소스를 다른 소스 위쪽에 놓아야 잘 보입니다. 소스 목록에서 드래그해서 순서를 바꿀 수 있어요.

## 6. 🔧 기능별 상세 설정

### 6.1 💬 채팅 반응

#### 이게 뭔가요?

AI가 시청자 채팅을 읽고 자동으로 반응하는 기능입니다. AmyayaBot의 핵심 기능이에요!

#### 어떻게 설정하나요?

반응 설정 탭에서:

| 설정 | 설명 | 기본값 |
|------|------|--------|
| 채팅 반응 간격 | AI가 몇 초마다 반응할지 | 10초 |
| 최소 반응 간격 | 반응과 반응 사이의 최소 대기 시간 | 6초 |
| 최대 응답 길이 | AI 응답의 최대 글자 수 | 50자 |

> 💡 **타이밍 프리셋**으로 한번에 조절할 수 있어요:
> - 패시브: 반응이 느림 (10초 이상 간격)
> - 균형: 적당한 속도 (기본 추천)
> - 액티브: 반응이 빠름

#### 사용 예시

> 시청자 "오늘 게임 뭐 해요?" 라고 채팅 → AI가 "오늘은 마인크래프트 하는 날이에요!" 같은 식으로 채팅방에 반응하고 말풍선으로 표시됩니다.

---

### 6.2 💝 후원 반응

#### 이게 뭔가요?

시청자가 후원(도네이션)을 하면 AI가 특별하게 반응해주는 기능입니다.

#### 어떻게 설정하나요?

후원 반응 탭에서:

1. **최소 반응 금액** 설정 (이 금액 이상 후원에만 반응)
2. **금액별 티어** 설정:
   - 예: 1,000원~5,000원 → 일반 반응
   - 예: 10,000원 이상 → 특별 반응 (TTS + 말풍선 + 긴 응답)
3. 각 티어별로 TTS 사용 여부, 말풍선 사용 여부를 따로 설정 가능

#### 사용 예시

> 시청자 "5,000원 후원! 재미있게 봤어요~" → AI가 "와 감사해요!! 정말 기뻐요!! 앞으로도 잘 부탁드려요 💕" 하고 음성으로 반응

---

### 6.3 🎙️ 음성 인식 (STT)

#### 이게 뭔가요?

STT는 "Speech To Text"의 줄임말로, 스트리머의 말소리를 글자로 변환하는 기능입니다. 이름을 부르면 AI와 대화할 수 있게 됩니다!

#### 어떻게 설정하나요?

STT 설정 탭에서:

1. **엔진 선택**:
   - **SenseVoice**: 빠르고 가볍습니다. 처음이라면 이걸 추천해요!
   - **Faster-Whisper**: 더 정확하지만 무겁습니다 (컴퓨터 사양이 좋아야 해요)
2. **마이크 선택**: 사용 중인 마이크를 드롭다운에서 선택합니다
3. **모델 다운로드**: 처음 한 번만 다운로드합니다 (시간이 걸릴 수 있어요)
4. **음성 레벨 테스트**: 마이크에 말하면 실시간으로 입력이 감지되는지 확인

> ⚠️ 모델 다운로드는 처음 한 번만 하면 됩니다. 인터넷 속도에 따라 수 분이 걸릴 수 있어요.

---

### 6.4 📣 호명 감지 & 대화 모드

#### 이게 뭔가요?

STT가 켜진 상태에서 AI의 이름을 부르면 "대화 모드"에 들어갑니다. 대화 모드에서는 AI와 1:1로 이야기를 나눌 수 있어요!

#### 어떻게 설정하나요?

반응 설정 탭 → 호명 감지 항목:

| 설정 | 설명 |
|------|------|
| 트리거 이름 | AI가 반응할 이름 목록 (예: 에이미, 아야, 아먀야) |
| 대화 최대 턴 수 | 몇 번 주고받고 대화 모드를 종료할지 (기본 10턴) |
| 자동 종료 시간 | 이 시간 동안 말이 없으면 대화 모드 자동 종료 (기본 30초) |

#### 사용 예시

> 스트리머가 "아먀야, 요즘 시청자들이 어떤 것 같아?" 라고 말함
> → AI가 "요즘 채팅 분위기 정말 좋은 것 같아요! 다들 신나 보이더라고요~" 하고 대답
> → 이후 계속 대화 모드로 이야기 가능

---

### 6.5 🔊 TTS 음성

#### 이게 뭔가요?

TTS는 "Text To Speech"의 줄임말로, AI의 텍스트 반응을 음성으로 변환해주는 기능입니다.

#### 어떻게 설정하나요?

TTS 음성 탭에서:

1. **엔진 선택**:
   - **Edge-TTS**: 무료, 인터넷 연결이 필요합니다 (Microsoft 서버 이용)
   - **Supertonic**: 내 컴퓨터에서 실행 (인터넷 없이도 가능, 더 자연스러운 목소리)
2. **음성 프리셋 선택**: 기본으로 10개의 음성이 제공됩니다
3. **속도, 음높이, 볼륨** 슬라이더로 조절
4. **"미리 듣기"** 버튼으로 테스트 후 마음에 드는 설정 저장

#### 사용 예시

> Edge-TTS 엔진, 여성 음성, 속도 1.1배로 설정하면 약간 빠르고 경쾌한 목소리로 반응합니다.

### 6.6 💬 말풍선

#### 이게 뭔가요?

AI의 반응 텍스트를 방송 화면에 예쁜 말풍선으로 표시합니다. OBS 오버레이와 연동됩니다.

#### 어떻게 설정하나요?

출력 설정 탭에서:

| 설정 | 설명 |
|------|------|
| 위치 | 말풍선이 나타날 방향 (오른쪽/위/아래/왼쪽) |
| 크기 | 말풍선의 전체 크기 |
| 폰트 크기 | 텍스트 크기 |
| 배경색 | 말풍선 배경 색상 |
| 글자색 | 텍스트 색상 |
| 테두리 색 | 말풍선 테두리 색상 |
| 표시 시간 | 말풍선이 화면에 머무는 시간 (기본 5초) |

### 6.7 🛡️ 콘텐츠 관리

#### 이게 뭔가요?

AI가 특정 주제에 대해 반응하지 않도록 필터링하는 기능입니다. 방송 사고를 예방할 수 있어요.

#### 어떻게 설정하나요?

콘텐츠 관리 탭에서:

| 필터 | 설명 |
|------|------|
| 정치 | 정치적 발언, 특정 정치인 언급 등을 차단 |
| 종교 | 특정 종교에 대한 내용 차단 |
| 성적 표현 | 선정적인 내용 차단 |
| 욕설 | 욕설 및 비속어 관련 반응 차단 |
| 차별 | 혐오, 차별 발언 관련 내용 차단 |

각 항목을 **ON/OFF**로 개별 설정 가능합니다.

**커스텀 차단 키워드 추가**: 직접 특정 단어를 입력해서 그 단어가 포함된 채팅에는 반응하지 않도록 설정할 수 있습니다.

## 7. 🎮 음성 명령 & 채팅 명령어

### 7.1 🗣️ 음성 명령

#### 이게 뭔가요?

STT가 켜진 상태에서 이름을 부른 뒤 명령을 말하면 AI가 실행해줍니다. 마우스나 키보드를 쓰지 않아도 방송을 제어할 수 있어요!

#### 어떻게 사용하나요?

1. STT가 켜져 있어야 합니다 (빠른 설정에서 확인)
2. **"[AI 이름], [명령]"** 형태로 말합니다

#### 사용 예시

| 말하는 것 | AI가 하는 것 |
|-----------|-------------|
| "아먀야, 투표 시작해 A는 치킨 B는 피자" | 채팅 투표 시작 |
| "아먀야, 추첨해줘" | 최근 채팅 참여자 중 1명 추첨 |
| "아먀야, 좀 조용히 해" | 반응 간격을 늘림 |
| "아먀야, 채팅으로만 대답해" | 말풍선/TTS를 끄고 채팅만 출력 |
| "아먀야, 채팅 요약해줘" | 최근 채팅 내용 요약 |
| "아먀야, 노래 신청 열어줘" | 노래 신청 큐 활성화 |

---

### 7.2 💬 채팅 명령어

#### 이게 뭔가요?

시청자가 채팅창에서 `!`로 시작하는 명령어를 입력하면 AI가 실행해줍니다.

#### 기본 명령어 목록

| 명령어 | 설명 |
|--------|------|
| `!투표 A` | 진행 중인 투표에 A로 참여 |
| `!투표결과` | 현재 투표 결과 확인 |
| `!참여` | 추첨에 이름 등록 |
| `!노래 곡이름` | 노래 신청 (예: `!노래 봄날`) |
| `!큐` | 현재 노래 신청 큐 확인 |
| `!스킵` | 현재 곡 스킵 (스트리머만 사용 가능) |

#### 설정 방법

- **빠른 설정 → 🎮 상호작용** 에서 채팅 명령어 기능 ON/OFF
- `!` 접두사는 설정에서 다른 기호로 변경 가능합니다

## 9. 🆘 문제 해결

### ❌ 프로그램이 시작되지 않아요

**확인 사항:**

1. Python이 설치되어 있는지 확인
   - 명령 프롬프트(cmd)에서 `python --version` 입력
   - 버전 번호가 나와야 합니다
2. (소스 실행 시만 해당) Node.js가 설치되어 있는지 확인 — 빌드 버전에서는 불필요
   - 명령 프롬프트에서 `node --version` 입력
3. **Windows**: `start.bat`를 마우스 오른쪽 클릭 → **"관리자 권한으로 실행"** 해보세요

---

### ❌ 아바타가 보이지 않아요

**확인 사항:**

1. OBS 브라우저 소스 URL이 `http://localhost:18300/overlay` 인지 확인
2. AmyayaBot 프로그램이 실행 중인지 확인
   - 브라우저에서 `http://localhost:18300` 에 접속해보세요
   - 페이지가 열리면 백엔드 서버가 실행 중입니다

---

### ❌ AI가 채팅에 반응하지 않아요

**확인 사항:**

1. **API 연결 탭**에서 Gemini API 키가 올바르게 입력되어 있는지 확인
2. 치지직 연결 상태가 **"연결됨"** 인지 확인
3. **빠른 설정**에서 **"시청자 채팅에 반응해줘"** 가 ON인지 확인
4. **빠른 설정**에서 서비스가 시작된 상태인지 확인 (서비스 제어 영역)

---

### ❌ 목소리(TTS)가 나오지 않아요

**확인 사항:**

1. **빠른 설정**에서 **"목소리로 대답해줘"** 가 ON인지 확인
2. **TTS 음성 탭**에서 음성 프리셋이 선택되어 있는지 확인
3. Edge-TTS 사용 중이라면 **인터넷 연결** 확인
4. 컴퓨터 볼륨이 켜져 있는지, 음소거가 아닌지 확인

---

### ❌ STT(음성 인식)가 동작하지 않아요

**확인 사항:**

1. **빠른 설정**에서 **"내 목소리를 듣게 해줘"** 가 ON인지 확인
2. **STT 설정 탭**에서 올바른 마이크가 선택되어 있는지 확인
3. 모델이 다운로드되어 있는지 확인 (STT 설정 탭에서 확인 가능)
4. 음성 레벨 테스트에서 마이크 입력이 감지되는지 확인

---

### ❌ Windows에서 설치 중 에러가 나요

**확인 사항:**

- **Python 3.12 또는 3.13** 사용을 권장합니다
- Python 3.14에서는 일부 패키지 설치 시 에러가 발생할 수 있습니다
- 아래 방법을 시도해보세요:
  1. **Microsoft C++ Build Tools** 설치
     - https://visualstudio.microsoft.com/visual-cpp-build-tools/ 에서 다운로드
     - "Desktop development with C++" 항목 선택 후 설치
  2. 설치 후 `start.bat` 다시 실행

## 10. 💡 팁 & 트릭

### 처음 시작할 때

- 타이밍 프리셋을 **"패시브"** 로 시작하세요. 반응이 너무 많으면 방송이 어수선해질 수 있습니다.
- 방송을 몇 번 해보면서 반응 속도와 빈도를 조금씩 늘려가세요.

### 설정 저장 잊지 마세요!

- 설정을 변경한 후에는 반드시 **"설정 저장"** 버튼을 눌러야 적용됩니다.
- 저장하지 않고 페이지를 닫으면 변경 사항이 사라질 수 있습니다.

### 프로필 활용하기

- 방송 유형별로 프로필을 미리 만들어두면 방송 시작 전에 빠르게 전환할 수 있어요.
- 새 설정을 시험해볼 때도 프로필로 저장해두고 테스트하세요.

### AI 봇 구분하기

- AI가 채팅방에 메시지를 보낼 때 **[캐릭터이름]** 접두사가 붙습니다.
- 이를 통해 일반 채팅과 AI 봇의 응답을 쉽게 구분할 수 있습니다.

### 페르소나 설정 잘 활용하기

- 프리셋을 먼저 적용한 후 세부 설정을 조정하면 더 편합니다.
- 스트리머 이름과 방송 분위기를 상세히 적을수록 더 자연스러운 AI 반응을 얻을 수 있어요.

### OBS 설정 팁

- OBS에서 "AI 봇 오버레이" 소스를 장면의 **맨 위**에 놓아야 잘 보입니다.
- 화면 밖으로 아바타가 나가면 OBS에서 소스를 선택하고 드래그로 위치를 조정하세요.
